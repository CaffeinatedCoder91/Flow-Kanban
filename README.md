# Jojo Todo App

Simple React + Express + SQLite todo app.

## Setup

```bash
npm install
```

## Development

Start both frontend and backend:

```bash
npm run dev
```

Frontend: http://localhost:5173
Backend: http://localhost:3001

## Reset Database

```bash
npm run db:reset
```

Or manually delete `db/todos.db`.

## Tests

```bash
npm test
```

Watch mode:

```bash
npm run test:watch
```

## Lint

```bash
npm run lint
```
